# Small Bees
Small Bees MCBE Add-On reduces the size of bees and other small animals to make them more realistic.

Besides bees, this add-on also reduces the size of axolotls and frogs by 50%. It does not modify any other aspect of these animals—their behaviors, health, and damage remain unchanged.

This addon is optimized to run effectively on Realms and dedicated servers with minimal hardware resource consumption ⚡
